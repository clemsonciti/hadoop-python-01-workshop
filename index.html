Introduction to Linux

Learning objectives

By the end of this workshop, you will be able to

  1. Create, manage and navigate files and directories on the linux filesystem.
  2. Chain together linux commands to perform complex tasks such as sorting and filtering.
  3. Write loops to perform repetitive tasks.
  4. Write shell-scripts to automate tasks.

Prerequesites

No prior experience with the Linux or command-line is required.
